Varnish One Liners

Aug 18th, 2014
  • devops
  • Fetch varnish logs for particular IP address: varnishlog -o -c | perl -ne 'BEGIN { $/ = "";} print if (/RxURL.*\/$/m and /ReqStart.*127.0.0.1/);'
  • Fetch 50x errors from backend (Varnish < v3): varnishlog -c -o TxStatus ^50
